# PaperFetch Pro 🔬

**A free, open-source research paper search engine — no API key, no signup, just results.**

🌐 **Live Demo:** `https://YOUR-USERNAME.github.io/paperfetch/`

---

## Features

- **9 Real Databases** — CrossRef (100M+), OpenAlex (250M+), PubMed, Europe PMC, DOAJ, PLOS ONE, Semantic Scholar, arXiv, CORE
- **No API Key Required** — all free public APIs
- **Journal Filtering** — Physics of Fluids, Journal of Energy, Nature, Science, Applied Energy & more
- **Export** — JSON, CSV, BibTeX, RIS (for Zotero / Mendeley / EndNote)
- **Works offline** — single HTML file, no server needed
- **Open Access PDFs** — direct PDF links where available

## How to Deploy on GitHub Pages

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set Source to **main branch / root**
4. Your site will be live at `https://YOUR-USERNAME.github.io/paperfetch/`

## How to Use Locally

Just download `index.html` and open it in any browser. No installation needed.
